Vc Project ISM330DHCX Support

Library support for project functionality.

Written by E. Lindsey Enders for LCAS

https://lcas.ece.utah.edu/

Library includes:
Accelerometer support for XYZ axes.
Gyroscope support for XYZ axes.
Temperature support.
Accelerometer and Gyroscope range changes.
Accelerometer and Gyroscope data rate changes.

Also supports print status for debugging to serial monitor.
